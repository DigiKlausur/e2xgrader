Klausur abgeben¶

Die Abgabe der Klausur besteht aus zwei Schritten.

  1. Abgabe

  2. Überprüfung der Abgabe

Abgabe der Klausur¶

1. Speichern Sie Ihre Klausur¶

2. Geben Sie Ihre Klausur ab¶

3. Überprüfen Sie den Zeitstempel¶

Überprüfung der Abgabe¶

Sie können Ihre Abgabe nach dem Klick auf Submit überprüfen. Öffnen Sie hierfür die generierte HTML Datei.

1. Überprüfen Sie die HTML Version Ihrer Abgabe¶

In der HTML Datei sollten alle Ihre Antworten stehen.

2. Überprüfen Sie den Zeitstempel im Notebook¶

Am Ende der Seite finden Sie den Zeitstempler (Timestamp) Ihrer Abgabe.

Stellen Sie sicher, dass der Zeitstempel korrekt ist. Aktualisieren Sie das Browserfenster falls nötig.¶

Unter dem Zeitstempel finden Sie den Hashcode. Sollten Sie diese Datei nicht öffnen können, finden Sie den Hashcode auch in der Textdatei (siehe nächster Abschnitt).

3. Überprüfen Sie die Textdatei mit Ihrer Prüfsumme (Hashcode)¶

Navigieren Sie zur Textdatei mit dem Hashcode. Der Name der Datei ist Ihr Benutzername, gefolgt von _info.txt (bspw. tmetzl2m_info.txt).

4. Füllen Sie den Zettel für die Aufsicht aus¶

Notieren Sie den Hashcode und Zeitstempel (HH:MM:SS) auf dem Zettel für die Aufsicht.Wenn Sie sich in einem Laptop Poolraum befinden, notieren Sie zusätzlich die Nummer des USB Sticks und des Laptops auf dem Zettel.

5. Füllen Sie den Zettel zum Mitnehmen aus¶

Notieren Sie den Hashcode und Zeitstempel (HH:MM:SS) auf dem Zettel, welchen Sie mitnehmen.
